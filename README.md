# CalcSuite — Financial Calculator Suite MVP

A comprehensive, privacy-first financial calculator suite built with Next.js 14, Tailwind CSS, and Chart.js. 15+ interactive tools for mortgage, loans, budgeting, and retirement.

## About 
**CalcSuite** is a modern, high-traffic financial utility suite designed for speed and user privacy. It provides 15+ deterministic financial tools—from mortgage amortization to debt payoff strategies—all calculated entirely on the client side.

**Key Highlights:**
- **Privacy First:** No sign-ups or server-side data storage. Data never leaves the browser.
- **Modern Stack:** Built on Next.js 14 (App Router) with TypeScript, Tailwind CSS, and Chart.js.
- **SEO Optimized:** Automated JSON-LD schema injection for search engine rich snippets.
- **Calculators Included:**
  - Mortgage Calculator
  - Loan Calculator
  - Compound Interest
  - Debt Payoff (Avalanche/Snowball)
  - Budget Analyzer (50/30/20)
  - Retirement Savings
  - ROI & CAGR Analyzer
  - Credit Card Payoff
  - Savings Goal
  - Simple Interest
  - Inflation Impact
  - Net Worth Tracker
  - Hourly-to-Salary Converter
  - Emergency Fund
  - Break-Even Analysis

## Getting Started

First, install dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```
